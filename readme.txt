## Libraries Used
-Pygame zero
-math
-random
-Rect for Pygame

## installation and setup
1.downlado the Game file
2.Make sure you have python is Installed already if not go to python.org and install it
3.Make sure you have pygame zero is installed already if not open you terminal and write pip install pgzero

## How to run
open terminal and type python.main

## in-game controls
WASD: Movement Player
Mouse Movement: aim shooting direction
Left click: Shooting
Space (Game over screen): return to main menu

## Development note
If you want to chagne music you need to make sure the background or boss music must be in OGG or MP3 format otherwise it won't work.
